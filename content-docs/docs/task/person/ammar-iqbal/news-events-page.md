## Task Description

Add a News & Events page to [https://roboconinc.com/](https://roboconinc.com/)

## Requirements

### Design Reference
- Similar to: [https://canvas.build/latest-news/](https://canvas.build/latest-news/)

### Technology Stack
- **Frontend**: React + Vite (same as current site)
- **Backend**: Supabase for data storage

### Database Schema

Create a new `news_events` table in Supabase with the following columns:

- `id` - Primary key
- `title` - News article title
- `summary` - Short summary/description
- `content` - Full article content
- `image_url` - URL to featured image
- `published_at` - Publication timestamp
- `category` - News category
- `author` - Author name
- `slug` - URL-friendly identifier
- `is_featured` - Boolean for featured articles
- `created_at` - Creation timestamp
