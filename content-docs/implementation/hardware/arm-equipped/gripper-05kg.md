# Arm Equipped: Gripper 05kg

5kg payload gripper system for robotic arm applications.

## Overview

The Gripper 05kg system provides a 5kg payload capacity gripper integrated with RoboCon Cobot 5kg arm, including depth camera integration and finger gripper components.
