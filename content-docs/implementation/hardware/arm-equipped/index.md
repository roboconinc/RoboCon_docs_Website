# Arm-Equipped Systems

This section contains hardware implementation details for RoboCon robotic arms equipped with various end effectors and tools.

## Overview

Arm-Equipped systems are robotic arms integrated with specialized end effectors, tools, and accessories for specific construction and industrial tasks. Each system includes the arm assembly, connectors, end effectors, and supporting documentation.

## Asset Folder Structure

Each Arm-Equipped system folder contains:

- **`/Tasks`** - Task specifications and requirements
- **`/{{YYYY-MM-DD}}_{{contractor}}`** - Deliverable folders with:
  - `/Drawings` - Technical drawings
  - `/Renderings` - Visual renderings
  - `/Solidworks` - CAD files
  - `/STEP` - STEP format files
  - `/STL` - STL format files
  - `/3D` - Other 3D file formats
  - `/Docs` - Documentation
