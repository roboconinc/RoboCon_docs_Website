# Robotic Arms

Information about robotic arm systems used in RoboCon robots.

## Overview

Robotic arms are essential components of RoboCon robot systems, providing manipulation capabilities for various tasks including handling, assembly, and bimanual operations.

## Dual-Arm Systems

### Servicer Tracked 15kg / Servicer Wheeled 15kg

The RoboCon Servicer models use dual 6-DOF robotic arms for bimanual manipulation.
