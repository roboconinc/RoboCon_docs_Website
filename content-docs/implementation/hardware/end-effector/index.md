# End Effector & Attachments

This section contains hardware implementation details for RoboCon end effectors, connectors, attachments, and related components.

## Overview

End Effector systems include grippers, connectors, crane attachments, fittings, pumps, switches, and other components used with RoboCon robotic arms and crane systems. Each asset includes specifications, documentation, and implementation details.

## Available End Effector Assets

This section documents all end effector assets from the End Effector project folder.

## Asset Folder Structure

Each End Effector asset folder contains:

- **`/Docs`** - Documentation files (PDF, DOC, DOCX) containing asset information, specifications, and technical details
- **`/Tasks`** - Task specifications for contractors
- **`/{{YYYY-MM-DD}}_{{contractor}}`** - Deliverable folders with:
  - `/Drawings` - Technical drawings
  - `/Renderings` - Visual renderings
  - `/Solidworks` - CAD files
  - `/STEP` - STEP format files
  - `/STL` - STL format files
  - `/3D` - Other 3D file formats
  - `/Docs` - Documentation

For detailed specifications, technical drawings, and implementation details, refer to the individual asset pages.
