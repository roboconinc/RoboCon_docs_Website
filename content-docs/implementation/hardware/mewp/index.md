# MEWP (Mobile Elevated Work Platform)

This section contains documentation for Mobile Elevated Work Platform (MEWP) assets, including boom lifts, cranes, mast lifts, and scissor lifts.

## Overview

MEWP assets are organized by type:
- **Boom** - Articulating and telescopic boom lifts
- **Crane** - Spider cranes and mini cranes
- **Mast** - Mast lifts and vertical lifts
- **Scissor** - Scissor lifts

## Asset Folder Structure

All MEWP assets follow the naming convention: `MEWP_{Type}_{Name}`

Each asset folder contains:
- **`/Docs`** - Documentation files (PDF, DOC, DOCX) containing asset information, specifications, and technical details
- **`/Tasks`** - Task specifications for contractors
- Asset deliverable folders with date and contractor naming
