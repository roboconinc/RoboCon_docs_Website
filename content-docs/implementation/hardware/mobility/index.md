# Mobility Components

Information about mobility systems used in RoboCon robots.

## Overview

Mobility components are essential for RoboCon robot systems, providing locomotion capabilities including tracked and wheeled base systems, motors, gearboxes, and related components.

## Component Categories

### Tracked Systems

Tracked undercarriage systems and track components for tracked robot platforms.

### Wheeled Systems

Wheel components and wheeled mobility systems.

### Motors

Motor components and drive systems for mobility applications.

### Gearboxes

Gearbox components and reduction units for mobility systems.

### Motor Drivers

Motor driver controllers for mobility applications.

### AMR Systems

Autonomous Mobile Robot (AMR) systems and components.

## Asset Information

This section contains detailed information about mobility-related assets, including:
- Technical specifications
- Manufacturer documentation
- CAD files and drawings
- Integration information
